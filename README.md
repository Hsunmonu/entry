# Entry

**AI call copilot for real estate and beyond.**

Entry listens to your phone calls in real-time, transcribes them, generates summaries and action items, and pushes structured data into your CRM — eliminating manual note-taking and follow-up tracking.

## Download

**Windows:** [Download Entry Setup (v1.5.0)](https://github.com/Hsunmonu/entry/releases/latest/download/Entry-Setup-1.5.0.exe)

## Features

- Real-time call transcription with echo cancellation
- AI-generated summaries, action items, and follow-up emails
- CRM integrations: HubSpot, Pipedrive, Follow Up Boss, kvCORE, Notion, Slack, Google Sheets, Greenhouse, and webhooks
- Notes Mode for general meetings + Realtor Mode for real estate workflows
- Per-provider push badges — choose exactly which integrations receive your call data
- Desktop hotbar overlay for quick controls during calls
- Auto-end on inactivity with smart generation
